Hexo 博客操作指令大全

基础指令

初始化新博客

创建新博客

hexo init my-blog
cd my-blog

安装依赖

npm install

或使用 yarn

yarn install

文章管理

创建新文章

基础创建

hexo new “文章标题”

简写

hexo n “文章标题”

指定路径

hexo new “文章标题” –path categories/tech/article.md

指定布局

hexo new “文章标题” –layout draft # 草稿

hexo new “文章标题” –layout page # 页面

文章 Front-matter 示例

-–

title: 文章标题

date: 2025-11-21 14:30:00

updated: 2025-11-22 10:00:00

categories:

  • 技术

  • 前端

tags:

  • JavaScript

  • Vue

  • 教程

description: 文章描述

keywords: 关键词1, 关键词2

thumbnail: /images/thumbnail.jpg

comments: true

toc: true

mathjax: false

-–

管理文章状态

发布草稿

hexo publish draft-article

hexo p draft-article

渲染草稿（预览）

hexo generate –draft

hexo g –draft

服务器操作

本地服务器

启动本地服务器（默认端口4000）

hexo server

hexo s

指定端口

hexo server -p 5000

hexo s -p 5000

指定IP地址

hexo server -i 192.168.1.100

hexo s -i 0.0.0.0 # 允许外部访问

静默模式（不输出日志）

hexo server –silent

hexo s -s

部署发布

一键部署

yarn hexo clean && yarn hexo generate && yarn hexo deploy
